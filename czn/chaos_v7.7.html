카드 추가

각 캐릭터 박스의 카드 추가 버튼으로 중립 / 몬스터 / 금기 / 고유카드(복제 개념)을 넣을 수 있습니다.
고유카드 8장은 기본 제공되며, 여기서 추가하는 고유카드는 모두 복제된 고유카드 로 취급됩니다.

카드 규칙

  • 중립 카드: 기본 20pt, 제거 시 0pt
  • 몬스터 카드: 기본 80pt, 제거 시 0pt
  • 금기 카드: 기본 20pt, 제거 시 0pt
  • 고유(normal): 기본 0pt, 변환 시 중립카드 생성(20pt) + 변환 10pt
  • 고유(rare/legend): 신 번뜩임 20pt(번뜩임 0)
  • 고유(myth): 번뜩/신번 없음, 복제만 가능

제거 규칙

  • 제거 누적: 0 / 10 / 30 / 50 / 70 / 70 ... (누적합)
  • 고유카드 제거 시 +20 추가
  • 제거된 카드는 강화(spark/new) 효과가 사라짐

복제 규칙

  • 복제 누적: 0 / 10 / 30 / 50 / 70 / 70 ... (누적합)
  • 중립/몬스터 강화: 번뜩 +10, 신번 +30
  • 고유카드 복제: 신번 +20

변환 규칙

  • 고유 normal에서만 변환 가능
  • 변환 = 중립카드 자동 생성(20pt) + 변환 10pt
  • 변환 중에는 변환 OFF 불가 (중립카드를 삭제해야 해제)
